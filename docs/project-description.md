# Описание проекта

## Концепция

Сервис автоматизированных онлайн-уроков по химии. Виртуальный ассистент подключается к видеоконференции голосом реального преподавателя, объясняет материал по конспекту, управляет доской Miro и отвечает на вопросы ученика в реальном времени.

## Роли пользователей

- **Преподаватель** — управляет через веб-кабинет: база знаний, расписание, настройки голоса
- **Ученик** — участвует в конференции: слышит голос, видит доску, задаёт вопросы устно
- **Администратор** (опционально) — управляет аккаунтами, мониторит нагрузку

## Ключевые функции

1. Веб-кабинет преподавателя (React)
2. Иерархическая база знаний (Класс → Раздел → Тема → Файлы)
3. Календарь занятий с привязкой к Zoom / Яндекс Телемост
4. Бот-участник конференции (голос + демонстрация Miro)
5. Синтез речи голосом преподавателя (клонирование)
6. Распознавание речи ученика + диалог в реальном времени
7. RAG по базе знаний для корректных химических ответов
8. Запись урока и транскрипт

## Нефункциональные требования

- Задержка ответа бота: не более 4 секунд (90-й перцентиль)
- Корректность химических фактов: не более 1 ошибки на 10 уроков
- Автоматическое переподключение при сбоях сети
- Шифрование всех соединений
- Соответствие законодательству о персональных данных
