## Этапы реализации

### Этап 1 — Прототип
- [ ] Веб-интерфейс (статический, локальное состояние)
- [ ] Ручной запуск бота скриптом
- [ ] Виртуальный микрофон + базовый TTS
- [ ] Управление Miro вручную через API

### Этап 2 — MVP
- [ ] Backend API + PostgreSQL
- [ ] Авторизация преподавателей
- [ ] База знаний с хранением файлов в S3
- [ ] Автозапуск бота по расписанию
- [ ] Интеграция с Zoom SDK или эмуляция
- [ ] Ведение урока по конспекту

### Этап 3 — Продукт
- [ ] Клонирование голоса (ElevenLabs или on-premise)
- [ ] Сложные диалоговые стратегии (RAG + дообучение)
- [ ] Запись и транскрипт урока
- [ ] Аналитика и журнал
- [ ] Поддержка нескольких ВКС-платформ
