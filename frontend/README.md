# Frontend — Кабинет преподавателя

React-приложение для управления уроками, базой знаний и настройками бота.

## Требования

- [Node.js](https://nodejs.org/) версии 18 или выше — скачать с официального сайта, установить с настройками по умолчанию
- После установки открыть **командную строку** (Win+R → `cmd`) или **PowerShell** и проверить:

```cmd
node -v
npm -v
```

Оба должны вернуть номер версии — всё готово.

## Структура

```
src/
├── components/
│   ├── Dashboard/      # Дашборд: статистика, ближайшие занятия
│   ├── Calendar/       # Календарь занятий
│   ├── Lessons/        # Список и управление уроками
│   ├── KnowledgeBase/  # База знаний: папки, файлы, конспекты
│   └── shared/         # Переиспользуемые компоненты (Modal, Toast и др.)
├── pages/              # Страницы приложения
├── hooks/              # Кастомные React-хуки
├── store/              # Zustand стор (состояние приложения)
├── utils/              # Вспомогательные функции
├── styles/             # Глобальные стили, CSS-переменные
└── assets/             # Иконки, изображения
```

## Запуск для разработки

Открыть командную строку или PowerShell в папке `frontend`:

```cmd
npm install
npm run dev
```

После этого открыть в браузере: http://localhost:5173

> **Совет:** в проводнике можно открыть нужную папку, написать `cmd` прямо в адресной строке и нажать Enter — терминал откроется сразу там.

## Сборка для продакшна

```cmd
npm run build
```

Готовые файлы появятся в папке `dist\` — их можно положить на любой веб-сервер.

Чтобы проверить сборку локально:

```cmd
npm run preview
```
